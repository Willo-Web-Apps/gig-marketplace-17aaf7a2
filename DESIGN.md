# Design DNA — Tasq

## Visual Personality: bold-energetic
## Layout: community
## Typography: bold-display

### Color Palette
- Primary (Amber): #F59E0B → oklch(0.75 0.16 75)
- Accent (Near-black): #111827 → oklch(0.18 0.015 250)
- Surface (Warm cream): #FFFBF2 → oklch(0.99 0.015 90)
- Surface Dark: #1A1208 → oklch(0.11 0.025 75)
- Text: #111827 → oklch(0.18 0.015 250)

### Font Pairing
- Display: Archivo Black — 900 (massive hero type, section headlines)
- Body: Barlow — 400/500/600 (body text, UI labels, descriptions)

### Unique Design Choices
1. **Oversized "95%" numerical hero** — the fee percentage rendered at massive display size as a visual anchor, making the value prop viscerally obvious at a glance
2. **Alternating dark/cream section rhythm** — hero uses near-black background to feel like a rallying cry; content sections use warm cream; this directly breaks from all-white competitor look
3. **Stark comparison table** — a dark-background "Why Tasq?" section with side-by-side platform fee comparison using bold amber highlights to visually devastate competitor pricing
4. **Community impact counter bar** — large amber numerals on dark surface showing total worker earnings kept, tasks completed, and average savings; feels like a movement dashboard
